Radeon ProRender for Autodesk® 3ds Max®

Using 3ds Max® Standard Lights

3ds Max® Standard Lights and their parameters are not necessarily meaningful in the context of providing light for a ray traced scene.  It is possible to use Standard Lights with Radeon ProRender, as many scenes that have already been created may have included Standard Lights, however, these will effectively be interpreted as physical lights for the purpose of producing a ray traced render.

Here we'll look at the Standard Lights and see how they may vary when used in an Radeon ProRender scene.

Omni Light

When adding and Omni Light to a scene you can see that it is very bright.  The intensity will need to be reduced substantially for it to be useful.  Add an Omni Light and reduce the +Intensity/Color/Attenuation -> Multiplier to around 0.02 and the Color to blue.

Because Radeon ProRender treats the Standard Omni Light as a physical light, and is ray tracing the outcome, there will always be shadows.  Turning the Shadows check box off will have no effect as to whether shadows are displayed or not.  They will always be there.

Along with Shadows the other parameters that are not supported for Standard Omni Lights include:

  • +Decay

  • +Advanced Effects

  • +Shadow Parameters

  • +Shadow Map Params

  • +Atmospheres & effects

  • +Indirect Illumination

  • +Light Shader

Many of these effects that have to be specified for a Standard Omni Light are already inherited by the way the physical light is rendered by AMD FiIreRender.

Spot Light

Change the +General Parameters -> Light Type type to a Spot Light and change the color to something less dramatic like a beige or orange.  Rotate the Spot Light so that is pointing a convenient direction.  You can adjust the +Spotlight Parameters as well like Hotspot/Beam and Falloff/Field which are supported by Radeon ProRender.

 Like the Omni Light shadows and other parameters do not do anything as the Spot Light is again treated as a physical light.

Directional Light

The Directional Light is treated by Radeon ProRender like the sun.  Which means that although the Directional Light has an area within 3ds Max® it will not be rendered as an area.  The only meaningful parameters for the Directional Light are:

  • Direction

  • Intensity (Multiplier)

  • Color

If you change the +General Parameters -> Light Type type to a Directional Light you will see this effect.  Increase the +Intensity/Color/Attenuation -> Multiplier and you can see how it lights the whole scene as if it were being lit by the sun.  Below we have changed the 3ds Max viewport mode to from [Shaded] to [Realistic] to illustrate this.

Like the Omni Light and Spot Light shadows and other parameters do not do supported as the Directional Light is again treated as a physical light.
