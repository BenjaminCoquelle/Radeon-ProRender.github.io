Material Library

Included within the Radeon ProRender install is the addition of Radeon ProRender materials to the 3ds Max® material library for you to use.  

AMD Radeon ProRender Materials Breakdown:

  • Radeon ProRender Blend - Used to blend 2 or more materials together.

  • Radeon ProRender Diffuse - Basic color material,it can take bitmap files with 2D textures.  Has no reflectivity.

  • Radeon ProRender Diffuse Refraction - Same as Radeon ProRender Diffuse but has refractive option.

  • Radeon ProRender Emmisive - Used to generate mesh lights.

  • Radeon ProRender Microfacet - Similar to Radeon ProRender Diffuse but this material has reflective option by means of Roughness.  This can generate anything between a smooth reflective material and a matte material by changing the roughness amount.

  • Radeon ProRender Microfacet Refraction - Same as Radeon ProRender Microfacet but has refractive option.

  • Radeon ProRender Oren-Nayar - Used to generate a wide range of natural surfaces, such as concrete, plaster, sand, etc.

  • Radeon ProRender Reflection - Used to generate only reflection.  Will be used with Blend material to make more advanced materials such as glass and water.

  • Radeon ProRender Transparent - Used to generate only transparency .Will be used with Blend material to make more advanced materials such as meshes and layered shaders.

  • Radeon ProRender Ward - Used to generate metallic materials.  It also has Roughness and Normal map options.

These can be accessed through the Slate Material Editor Material/Map Browser.

You can preview any of the materials within the Slate Material Editor Material/Map Browser by expanding the + sign under each Material category.  Each preview must be generated from the material so it can take some time for all of them to be visible.

You can also view them as part of the Download suite where there are high resolution renderings of all of these complex materials.  Download the FR.CATALOG.4-1.6.41.22-2016_11-13-17_PM.zip  file, or similar as the version numbers may change over time, from the download page.  

Unzip the contents and double click the index.html file to see a browser layout of all the materials.  Double click any of the images within the browser to see the high resolution image.
