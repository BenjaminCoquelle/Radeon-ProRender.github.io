Assigning Basic Materials

To assign a material to an object from the Material Editor select your object or models in the scene then right click over Radeon ProRender Material you have previously dragged into the Material Editor workspace and select Assign Material to Selection to your selection.

You can also assign the pre-configured materials provided by Radeon ProRender in a similar way.  These materials combine maps and node connections to create a large library of different materials that you can use or modify to create just about any surface you desire.

Below we have taken the Objects_Demo.max scene provided with the samples file Test_Scene_Objects-2016-04-18.zip on the Download page and removed the materials from the glass and the liquid inside the glass.   Removing materials can be achieved by selecting the objects and typing $. mat = null in the max script command line or you can leave the current material assigned and overwrite it with the procedure blow.

To assign one of the Radeon ProRender pre-configured or blend library materials.  Open the Slate Material Editor as instructed in Display the Material Library and scroll down through the Material/Map Browser list to the material you would like to use.  Below we are going to choose the Glass_Antique (Radeon ProRender Blend Material).  Drag the selected material to the Material Editor panel.  You should now see how the material is constructed using other nodes.

Select the glass object in the scene then right click the right most material box.  In this case it is the Glass_Antique blend node.  Select the Assign Material to Selection from the popup menu that appears.

Now let's see if we can assign a material to the liquid object in the glass.  Previously it has a Liguids_Alcohol_Champagne material assigned.  In the next chapter lets see if we can turn this material into red wine which will go nicely with the antique green glass material we just assigned to the glass.
