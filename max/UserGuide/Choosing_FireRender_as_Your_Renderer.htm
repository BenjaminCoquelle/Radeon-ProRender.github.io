Choosing Radeon ProRender as Your Renderer

Render Setup

Once 3ds Max® has loaded close the Welcome Screen if 3ds Max® shows it.  This will leave you with a new scene.

Select the Render Setup button, Rendering -> Render Setup... from the menu or press the F10 key.

This will display the Render Setup dialog where you can set the default render you want to use for production or final rendering of your scene.  You will see that the Render is set to the Default Scanline Renderer or one of the other renders if you have changed this setting before.  Ensure the Target drop down is set to Production Rendering Mode and then using the Render drop down select Radeon ProRender.

 

Once you have done this you will see the settings tabs change to include Radeon ProRender Settings.  If you access the plug-in Manager again under Customize -> Plug-in Manager, as illustrated in the previous chapter, you will see the plug-in status has changed to "loaded".

Setup Lighting

Without setting up the lighting properly to take advantage of Radeon ProRender, any production renders will not display the advanced features that AMD Radeon ProRender provides.  

For lighting you can use a variety of techniques including using:

  1. Emissive (self illuminated) materials and geometry

  2. An environment map with a High Definition Range Image (HDRI) file

  3. 3ds Max® photometric lights

  4. 3ds Max® IES lights <coming soon>

  5. Physical sun and sky system

  6. 3ds Max® standard lights can also be used however these are less ideal as they do not obey physical properties (for example they have no decay and their multiplier is invalid in a physical environment).  Thus renderings may even create fireflies (colored spots)

For a the best rendering outcomes we recommend you use above lighting options.

To set up the lighting in our example we'll assign an environment map using a High Definition Range Image (HDRI) file.  If we set up a plane with a teapot on top and produce a render we can see that the result is a very standard render with sharp shadows and highlights.

 

To assign and HDRI environment map, open the Environments and Effects dialog by either selecting Rendering -> Environment from the menu or by pressing the 8 key.

There are a number of sample scenes provided along with the Radeon ProRender plug-in.  A description of the download location can be found on the download page mentioned in the Install, License, and Run Radeon ProRender for 3dsmax chapter and by following this link Radeon ProRender Releases.  Here we will be using a HDRI file located within the Test_Scene_Objects-2016-04-18.zip package.

Within the package is an Objects_Demo directory containing the Studio-Chrome-v2.hdr HDRI file.  

Assign a HDRI image file to the Environment Map and tick Use Map.

When you press the Environment Map button you will be presented with the Material/Map Browser.  Select the Bitmap and then locate the Studio-Chrome-v2.hdr file under Objects_Demo directory wherever you have saved it after extracting it from the .zip sample package.

If you have built a test scene like the teapot above, perform a test render by either selecting Rendering -> Render from the menu, pressing the Render Production button or hitting Shift+Q and have a look at the results.  

You can see that the HDRI lighting has been applied to the scene resulting in multiple shadows and smoother highlights as Radeon ProRender performs the ray tracing.

The render will also display a Rendering status window.  The first render you perform may take a while to initialize however successive renders will not require the initiation so will display a lot quicker.  The Rendering status also shows you the number of passes executed by Radeon ProRender.  With increasing passes the render will improve in detail and quality.  You can Pause and/or Cancel the render at any time.

The sample files also contain a Objects_Demo.max scene file that includes this HDRI image as an Environment Map as well as additional physical lighting on a transparent glass.  This shows you what is possible once we apply some materials.  It would be worth while to load this scene, test out a render, and see how the HDRI image and physical lighting can affect the result.

Assign Basic Materials

To assign basic materials to Objects or Models in a scene open the Material Editor either by using the Rendering -> Material Editor -> Slate Material Editor menu option, clicking the Material Editor button or hitting the M key.

You will notice that a number of Radeon ProRender materials have been added for your use during the installation.  These are discussed in more detail in the following Material Library topic.

After the Material Editor opens, navigate to  the Radeon ProRender menu set on the left hand side of the Material Editor.  Click and drag the Radeon ProRender Reflective Material into the Material Editors workspace.

 

 

Select your object or models in the scene then right click over Radeon ProRender Reflection Material in material Editor workspace and select Assign Material to Selection.

Now we are ready to render another test shot, this time with materials assigned to your selected model.

Production Render

To perform a Production Render either select Rendering -> Render from the menu, press the Render Production button or hit Shift+Q and have a look at the results.  

Radeon ProRender will start rendering through your selected camera.

Render Settings and Fine Tuning

Open the Render Setup,navigate to Radeon ProRender Settings Tab.  Set the pass limit to 100. This will limit the amount of time it will take to render the frame, but this trades off with quality of the render.

Start another render.  This time the progress bar starts to count the amount of passes.  It will stop when it gets to 100 passes.
