­­Radeon ProRender for Autodesk® 3ds Max®

System Requirements

The installer will provide a basic check of your hardware and provide you with a warning if your hardware may not be able to run Radeon ProRender efficiently.  For example, a check of your GPU's ability to run OpenCL™ 1.2 is conducted.  

If your GPU cannot run OpenCL™ 1.2 then the rendering will be configured to run on your CPU.  This is important to note as using the CPU for rendering may impact your ability to run other core applications while rendering is taking place.

It is recommended that you review your system's hardware configuration and it's suitability to run Radeon ProRender so as to avoid performance and/or compatibility issues when you use the plug-in.

Recommended Minimum PC Specification:

  • Autodesk® 3ds Max®  2017 or

  • Autodesk® 3ds Max® 2016 with Service Pack 3+

  • A PC with Windows® 7, Windows® 8, Windows® 8.1 or Windows® 10 installed.

  • Hardware Support:

    • Runs on both GPUs and CPUs.  

    • OpenCL™ 1.2 required for GPUs.

  • For a system report and to find out if your hardware supports OpenCL™ 1.2, run GPU-CAP-Viewer.

  • Recommended Hardware:

    • AMD FirePro™ W or S series or Radeon™ graphics card of the “Southern Islands” generation or beyond.

    • Examples: FirePro™ W9100, W8100, W7100 GPUs; Radeon™ 7950 GPU, Radeon™ 7970 GPU, Radeon™ R280 GPU, Radeon™ R290X GPU, etc.

At the time of writing, earlier versions of Autodesk® 3ds Max® to 2016 are not supported so it is important that you have the latest version of Autodesk® 3ds Max® 2016 Service Pack 3+ to ensure compatible operation of the Radeon ProRender plug-in.

In the future it is intended to support Autodesk® 3ds Max® 2014, 2015 and 2017 so stay tuned for news and updates as to when support for these versions becomes available.
