Radeon ProRender for Autodesk® 3ds Max®

Render Production

Set up Radeon ProRender as the Renderer.  Navigate to your Render Setup in the top right hand side of 3d Max.

 

Select Radeon ProRender in the Renderer drop down menu.

 

Go to the  fire render settings tab in the render setup,scroll down to hardware to see if Radeon ProRender is picking up the GPU, If you have dual GPU setup you can set it in the GPU#.  Or you can change the device to use the CPU.

 

Before changing any other settings, Navigate to the Render Production icon and do your first render.

It will start the render and show a progress bar.  At default your render setup will be set to 0 limits for the Pass limits which means it will keep calculating passes.

If you want to set the limits for the amount of passes to calculate during a render you can simply change the Pass Limit in the render setup.  I will be using a pass limit of 500 for this tutorial.
