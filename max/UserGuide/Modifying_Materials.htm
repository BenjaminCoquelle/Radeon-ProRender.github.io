Modifying Materials

Drag the Liguids_Alcohol_Champagne into the Material Editor panel below the glass material we just used.  This will allow us to see it better and change it.

You can see that the color of the liquid is set by a Color Map node which is fed through a Map node that splits it to be input into the Color Map connections of the Radeon ProRender Reflection and Radeon ProRender Refraction node.  This means if we change the color of the Color Map node we should be able to change the color of the liquid.

Double click the Color Map node/box.  This will bring up the color on the right hand side of the Material Editor.  Double click the Color and change the it to a deep burgundy/red.  

Select the liquid object within the scene and right click the right most node/box.  Select the Assign Material to Selection from the popup menu that appears.

Now render the scene to see the results by either selecting Rendering -> Render from the menu, pressing the Render Production button or hitting Shift+Q.
