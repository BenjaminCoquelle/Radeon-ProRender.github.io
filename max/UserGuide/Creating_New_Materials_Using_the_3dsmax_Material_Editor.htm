Creating New Materials Using the 3dsmax Material Editor

To create realistic looking materials we must first use our observation of surfaces in the real world to get an understanding of the different variations of color, reflection, bumps and imperfections that make objects stand out as having realistic looking surfaces.

Often real object surfaces have imperfections are effected by weather, use or even by their manufacturing process.  To make believable materials we need to add scratches and stains.  Most importantly, we need to provide a physical texture and rendering attributes which will effect the way light is reflected off the surface.

First we should select a surface we would like to use and add it to the Material Editor.  Below is an image of a metallic surface.  Right click the image and save it on your disk.

Once you have saved the file in a convenient place, locate the file with your file explorer and drag it onto the 3ds Max® Material Editor.

For a metallic surface we need a combination of roughness and shininess so we'll use the Radeon ProRender Microfacet Material.  Open the Radeon ProRender materials under the Materials tab in the Material/Map Browser. Select the Radeon ProRender Microfacet Material and drag it onto the Material Editor panel next to the Bitmap that you have already placed previously.

Right click the Radeon ProRender Microfacet Material node/box and select Open Preview Window.  This will allow you to view the material as you make changes to it.  Or you can double click the material box and size it to make the icon image larger.

You can see how this material combines shiny with rough by changing the Roughness to 0.0.  The material preview becomes perfectly shiny.  Change it back to 1 to make the material rough again.

To get the effect we desire the metal needs to be a darker color so we'll change the Color of the material to RGB 75, 75, 75.  Also change the Index of Refraction (IOR) for the Fresnel reflections to 1.51.  Remember to set the Roughness back to 1.0 as instructed above.

We could assign the Bitmap directly to the materials Color Map however this will make the material look flat when we would like a bumpy effect that looks like eroded or pitted metal.  Instead we will connect to Bitmap to the Normal Map using the Radeon ProRender Normal node which has the ability to use normal maps or bitmaps as input and translate the image into a bump map.

Find the Radeon ProRender Normal node under the Maps > Radeon ProRender tabs in the Materials/Maps Browser and drag it onto the Material Editor panel in between the Bitmap and the Radeon ProRender Microfacet material.  Connect the Bitmap to the Map connection on the Radeon ProRender Normal node and the output from the Radeon ProRender Normal node to the Normal Map connection on the Radeon ProRender Microfacet  material. Select the Radeon ProRender Normal node/box and make sure the Bump Map check box is ticked.

If we are using a Normal Map instead of a grey scale "Bump Map" we need to consider applying a Gamma Override to ensure we do not get stray artifacts on out resulting surface.

We can illustrate this by dragging a Bitmap into the Material Editor panel and opening a Normal Map.  You will be presented with a file dialog to select a Normal Map.  Navigate to the sample files and select Objects_Demo/Tiles_Normal_8x8.png.  Leave the Gamma set to Automatic which is the default.

Connect the Bitmap output to the Map node on the Radeon ProRender Normal material.  Set the Roughness on the Radeon ProRender Microfacet Material to 0.1 and expand the size of the material box to get a better view of the sample render or right click the Radeon ProRender Microfacet Material node/box and select Open Preview Window.  You will see that the bumps on the sample does not look right.  

Add a second Bitmap and open the same file but this time select the Gamma Override.

Connect the output of the new Bitmap to the Map of the Radeon ProRender Normal material.  You will see the preview now looks much better and has the bumps you would expect.  It is recommended whenever using Normal maps to set the Gamma Override when opening the file.

Delete these Bitmaps an put the materials back the way they were before.

Because we want some stains as well as bumps we are going to connect the Bitmap to the Roughness Map on the  Radeon ProRender Microfacet material as well.  You can see below how this changes the surface to give variation in the shininess and roughness.

We can now apply a couple of 3ds Max® standard nodes to help us control the effects or intensity of the bump and roughness.  To achieve this we can insert the Maps > Standard > ColorCorection map between the Bitmap and both the Radeon ProRender Microfacet node and the Radeon ProRender Normal node.  To do this simply drag the ColorCorection map onto the two red lines that connect the Bitmap to the other nodes/boxes.  

You can then adjust the Brightness and Contrast of these nodes to increase or decrease the intensity of the roughness/shininess and bumpiness of the metal material.

You can now apply the material to an object in your scene.  Select the desired object then right click over Radeon ProRender Microfacet material and select Assign Material to Selection.  Below we have assigned it to our teapot and made a test render to see what the final new metal material will look like.

To save this new material you can view all the materials used in a scene and either save all the materials into a new library or select a single material and save that into a separate library.  Open the Material Editor and scroll down to the Scene Materials tab in the Material/Map Browser and expand the + to show the materials underneath.  Right click the tab or one of the materials and select Save As New Material Library.

You will then be presented with a dialog to select which parts of your scene you want to extract the materials.  Select all of them for now.  Then choose the filename and where you want to save the library.

You library will then appear in the Material/Map Browser for you to use whenever you want.  Even if you open or create a new scene, the new material will be displayed in the Material/Map Browser for you to use.
