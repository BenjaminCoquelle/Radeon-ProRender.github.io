Install, License, and Run Radeon ProRender for 3dsmax

Downloading the Plug-in

To install Radeon ProRender plug-in you can go to the AMD website and select the developers branch of the website highlighted in red below.  

.

This will take you to the developers specific website which has many options for finding tools, SDKs, resources and the Radeon ProRender plug-in.  

From the main menu select the following Graphics & Game Development -> FirePro™ Developer Technologies -> AMD Radeon ProRender for Autodesk® 3ds Max®

 

This will take you to the following page which provides an overview of the Radeon ProRender capabilities and provides links to the download page and other resources like sample scenes and material library.

Scroll down past the overview and you will find links to the download page, material library, and sample scenes.  

Select the Radeon ProRender Max plug-in here link.  This will take you to a Github page where you will find the latest version of Radeon ProRender.  At the time of writing this was version 1.6.41.  There will be regular releases of Radeon ProRender as the beta program progresses.  These will be notified via an email which is provided when registering the product during installation covered below.  

Before downloading the plug-in installer it is advised to review your system requirements in the previous chapter System Requirements to ensure your system be able to run Radeon ProRender.

Select the Radeon ProRenderMaxplug-in_v1.n.nn_setup.exe to download the plug-in installer.  A link has not been provided here as this will likely change as new versions are released.

The download will take some time as the file is around 500MB. Depending on your internet connection the may be around 5 minutes.  Once the download is complete, run the file from the browser or download tool you have used.

Installation

Run the Radeon ProRenderMaxplug-in_vn.n.nn_setup.exe file.  Either locate the downloaded directory and double click the file or select run from with the browser download facility.

The installer should not require admin rights however Windows will ask if the .exe file can be trusted.  Press Yes to continue the install.

You will be asked to accept the licensing agreement for the software.  If you do not accept the install will complete without installing Radeon ProRender.  Please read the agreement and press I Agree to continue with the install.

The Installation Options will be presented and ask which components to install.   It is recommend that both be ticked.  The Runtime Environment option will install the Microsoft Visual C++ Redistributable for Visual Studio 2015 if it has not already been installed by another product.  Click Next to proceed.

The installer will then ask for a destination directory.  By default this is the C:\Program Files\AMD\Radeon_ProRender_Plug-ins\3DSMax directory.  If you wish to install the plug-in elsewhere you a free to change the directory location by hand or browse for a new location.  The installer will detect your 3ds Max® location automatically for applying the plug-in and materials.

You are now ready to install so press the Install button.

The install will quickly copy the required files to the required location.  If you wish to see those locations and make sure the installer is performing as expected you can press the Show Details button.

The install will pause at this point an display a new dialog.  The Installing dialog will remain open in the background.

A dialog will show you what versions of 3ds Max® you have installed as well as check your hardware configuration and GPU for the prerequisites to run Radeon ProRender.  

At this stage the pervious versions of 3ds Max® can be ignored as the current plug-in only supports 3ds Max® 2016.  You can check these versions and the installer will place the plug-in into that version's directory structure, however the plug-in will not run.

If your hardware is not compatible you will be presented with a warning that Radeon ProRender may not perform as expected if you chose to continue.

It is recommended that you only check the 2016 version for installation.  The press Next.

License

The installer will then present you with a licensing dialog where you can enter a licensing key which is provided to you via email on completion of the web based licensing form automatically displayed by the installer or you can follow the link highlighted by the dialog.    

The installer will open a web based Radeon ProRender licensing page using your default browser.  Complete the details within the form ensuring that the email you provide is active and you can access it to receive the license key.  It is also advisable that you check the spam folder, if you have a spam filter set up, in case the email is placed there by your email server and/or client.

This email will also be important in the future as it will be used to advise you of product updates.

Once you have completed your details press the Request Installation Key to receive the email containing the Radeon ProRender license key.  The Installation Dialog containing the field to enter the key will be active in the background.  Select the dialog and copy and paste the key from the email, once you receive it, and enter it into the space provided by the Installation Dialog. Then press Next.  You will receive a message in the status panel that the install has been successful so press Finish.

If you remember there is one other install dialog running in background.  The status bar will now have moved to the end and the status comment will say "Completed".  Press Close button to complete the installation.

You are now ready to open 3ds Max® to run and configure the Radeon ProRender renderer.

Run Radeon ProRender for 3ds Max®

To run the Radeon ProRender plug-in all you need to do is to start 3ds Max.  If you don't blink you may catch 3ds Max® loading the plug-in as it starts.  It will display the plug-ins that are loading in the load screen status line.  In the image below shows the status line with "Starting 3ds Max®..." because we blinked.

You can also check that the plug-in is available for 3ds Max® to use by accessing 3ds Max®'s plug-in Manager.  Under Customize -> Plug-in Manager.

Scroll down and you will be able to see the Radeon ProRender plug-in, the status, size and path of the plug-in file. The status above is set to "deferred" as the plug-in has not been set up as our default renderer yet.  Once we do this the status will change to "loaded".  We will be covering this in the next chapter.
