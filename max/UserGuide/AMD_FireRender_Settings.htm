Radeon ProRender for Autodesk® 3ds Max®

Radeon ProRender Settings

Radeon ProRender Settings

The first line of the settings is a version reference.  This allows you to identify which version of the plug-in you are running in case you run into difficulties and need to log a bug report.

Progressive Rendering Limits

Progressive render limits allow you control:

  • Pass limit, how many passes the render will be permitted before stopping. For example of set to 100 the render will complete after 100 passes.  

  • Time limit (HH-MM-SS], how long the render will go for before completion.  If set to 00-30-00 the render will run for 30 minutes then complete.

If none of these items are set the render will run indefinitely unless stopped with the Cancel button or paused with the Pause button.  A window is displayed along side the Production Render when a Render is started.  It displays the essential information about the render while allowing you to Cancel and/or Pause the render process.

Tone Mapping

As mentioned in the Working with Cameras chapter the Tone Mapping is a post render image processing effect that apples either:

  • Reinhard

  • Linear (Photographic Exposure)

Depth of Field

The Depth of Field (DOF) can be enabled for all cameras via the Enable check box.  

Radeon ProRender also supports the DOF settings on the Physical Camera which will enable DOF on camera by camera basis.  This is explained further in Working with Cameras.

Radeon ProRender Advanced Settings

Global Illumination

There are 2 Global Illumination method supported by Radeon ProRender:

  • Path Tracing

  • Cached GI

Clamped irradiance is set to 1 by default but can be changed if desired.

Performance Settings

Depending on the power of you CPU or GPU you may want to change the rendering mode to speed up rendering.  Modes supported include:

  • Global Illumination (GI)

  • Direct

  • Direct No Shadow

  • Wireframe

  • Diffuse albedo

  • Position

  • Normal

  • Tex coods

  • Ambient occlusion

There are 2 Quality Presets to help make the performance setting easier:

  • Production

  • Preview

Image Filter

A number of post render image filters are also supported including:

  • Box

  • Triangle

  • Gaussian

  • Mitchel

  • Lanczos

  • Blackmanharris

Trace Bounce Limits

If you want to limit the number of ray-trace light bounces that are calculated for the render, you can.  The default is unlimited but by selecting the Maximum Light Bounces option you can set the number of bounces.

Hardware

The Radeon ProRender install will check you system for compatibility and determine if you GPU will support OpenCL.  If it does then Radeon ProRender will assign the renderer to the GPU.  You can however change this if you want.  You can choose to render on:

  • CPU only

  • GPU only

  • or Bother CPU + GPU

If you have more than 1 GPU you can select the number of GPUs to assign the render to via the GPU#  as well as the Medit Quality and VFB Refresh rate.

 

Diagnostics

If for some reason Radeon ProRender appears to be crashing 3ds Max® you can set the Trace Dump check box to produce 2 diagnostic files that can help when tracing down problems.  These are valuable when submitting a bug report as they can assist developers in finding where in the plug-in issues may be arising.  

The 2 files are saved under the Program Files/Autodesk®/3ds Max® 2016 install directory are:

  • logFR_data.bin

  • logFR.cpp

You can also export the model in the scene to a directory of your choice.
