Custom Lighting Using an Emissive Material

Another technique to create lights for AMD FireRender is to use the FireRender Emissive Material to create an object or geometry that is a source of light in itself.   

Create a cylinder and move, rotate and scale it so that it becomes a long cylinder "hanging" over the car.  This will simulate a large long florescent light once we put the emissive material on.  

Open the Slate Material Editor and drag the FireRender Emissive Material from the Materials -> FireRender tab in the Materials/Maps Browser onto the editor panel.  Click on the material and adjust the Multiplier to be around 50.0.  After selecting the cylinder, right click the material and select Assign Material to Selection.

If you have the ActiveShade render window open you will see the result immediately, otherwise start a Render to see the emissive lighting.
