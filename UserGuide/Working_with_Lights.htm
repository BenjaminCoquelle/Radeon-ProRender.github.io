AMD FireRender for Autodesk® 3ds Max®

Working with Lights

Photometric Lights vs Standard Lights

As AMD FireRender is a physical ray tracing renderer, it treats lights differently to a normal renderer.  Where a normal renderer simulates the effects of lighting, AMD FireRender actually treats a light a real source of light and traces the individual beams of light on and around the objects in the scene.

As a result, many of the parameters required for 3ds Max Standard Lights are not meaningful in the context of interpreting these lights for a ray traced scene.  It is possible to use Standard Lights with AMD FireRender, as many scenes that have already been created may have included Standard Lights, however, these will effectively be interpreted as physical lights for the purpose of producing a ray traced render.

It is recommend that you use Photometric Lights if you are creating a new scene or modifying a current scene to work well with AMD FireRender.

Initial Lighting of a Scene

Without setting up the lighting properly to take advantage of AMD FireRender, any production renders will not display the advanced features that AMD FireRender provides.  

For lighting you can use a variety of techniques including:

  1. Emissive (self illuminated) materials and geometry, like the headlights of our car below

  2. An environment map with a High Definition Range Image (HDRI) file

  3. 3ds Max photometric lights

  4. 3ds Max IES lights <coming soon>

  5. Physical sun and sky system

  6. 3ds Max standard lights can also be used however these are less ideal as they do not obey physical properties (for example they have no decay and their multiplier is invalid in a physical environment).  Thus renderings may even create fireflies (colored spots)

For a the best rendering outcomes we recommend you use above lighting options.

For the rest of this chapter we'll be using the car scene from the samples included on the Download page. Details on how to find the AMD FireRender web site and Download page can seen in the Install, License, and Run FireRender for 3dsmax chapter.  

Follow the procedure from Import Geometry in Chapter 2: FireRender for 3dsmax Tutorial to set up a new scene with a car.

We are now going to set up a background to view the lights when rendering.  Maximise the perspective viewport and then press L, or right click the [Perspective] tag in the top left hand corner and select Left, to view from the left.  Create a rectangle with rounded corners so the edge of the rectangle just touches the wheels of the car.

Convert the rectangle to an Editable Spline by right clicking the rectangle and select Convert To: -> Convert to Editable Spline.  Select the top and side of the rectangle including the corners and delete them using the Segment button under the rectangle +Selection to select the edges and the Delete button under the +Geometry attributes.

Increase the smoothness of the remaining corner by changing the Interpolation to Adaptive.  Change the viewport from Left to Perspective. Then add the Extrude modifier to the line and increase the Amount to create a wall behind the car.  Move the wall so it is in a nice position behind the car.

Then add some volume to the wall with the Shell modifier.  Apply a Standard material to the wall to give it some color.  Just add a Standard material to the Material Editor and drag the output onto the wall.

Using the procedure Render Setup in Choosing FireRender as Your Renderer set the render to AMD FireRender and do a test render.

Once you render the scene you will see we don't have any lighting.  The render is very dark except for the car head lights which have an emissive material assigned so will appear to glow.  We need to add an HDRI background to the environment using the same procedure as we did with Setup Lighting in the Choosing FireRender as Your Renderer.  Make sure this time however that you choose the Studio-Chrome-v2.hdr file from the Vehicle sample then select Open so you can adjust the intensity of the background.

Click the 16 bit/chan Linear (48 bpp) and change the values of the White Point so that all of the image appear to be purple.

Now our scene has some basic lighting to which we can add more lights to see the effect they have.  So we can get a better view of the affects of our changes we are going to use the render viewport in the Active Shade mode.  Change the Target dropdown in the Render Setup to ActiveShade Mode.

Start a render and you will see as you move around, or move an object in the scene, that the render viewport will be updated with the changes you make.

Save the scene.
