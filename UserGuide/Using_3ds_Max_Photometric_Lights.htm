Using 3ds Max Photometric Lights

Photometric Free Light

It is recommended for physically accurate rendering with AMD FireRender that we use photometric lights.   These light will provide the best result in terms of realistic lighting, reflections, refraction and shadows.

To add a photometric light select the Photometric drop down within the Light object tab.  To add a free light to the scene click the Free Light button and click within the scene to place the object.  Move the light around to a position with which you a happy.

If we display the ActiveShade render window that we set up in Working with Lights, we can see that without any modification that the light provides a softer more realistic result.

We can make a copy of this light and do some interesting things with the light setting to make the scene look even more natural.  To clone a copy of the light hold the Shift key down and drag a copy of the light to a new position.  Repeat this until we have 3 lights above the car.  Select the Instance option button for the type of clone you are making.

It may be easier to position the light by viewing the scene from the top. Right click on the [Perspective] in the top left of the viewport and select [Top] or push the T key.  

Go back to the perspective view by right clicking [Top] and selecting [Perspective] or pushing P.  Hold Alt and the Middle Mouse button to rotate the scene so the car and lights are visible from a perspective angle.  One of the things we can do is add a shape to the light and turn it into an Area light.

Display the light's settings under the Modify tab and change the Shape/Area Shadows -> Emit light from (Shape) drop down to a Rectangle. You will see that all the lights will change to a rectangle as the are instances and inherit the settings of the each other.

Rotate the front 2 lights so the face the car.  You can see the shadows in the ActiveShade render are very soft and natural.  The rear light we will make into a larger rectangle and have it reflected in the window of the car.  This will look like a ceiling light panel or sky light you might find in a car display or sales floor.

Rotate the front lights by right clicking each light and selecting Rotate.  Drag the rotation handles to angle the rectangle to face the car.  

Select the rear light and click the Make Unique button to separate it from the other lights.  Increase the Length and Width of the rectangle so it appears to be a large square.  Enable the Rendering -> Light Shape Visible in Rendering check box to show the rectangle in the ActiveShade render.

Move the light so it is over the car,  raise it up a bit so the light is not so close to the car, and we can get a shot without the light in the field of view.  Rotate the scene so we have a good view of the car and the reflection of the area light in the window and bonnet.

Light Distribution Type

AMD FireRender will support 4 distribution types.  These include:

  • Photometric Web (IES Lights) <Coming Soon>

  • Spot Light

  • Uniform Diffuse

  • Uniform Spherical

Uniform Spherical

As above a Uniform Spherical  setting on the Light Distribution Type will cast light in all directions.  For a light with a rectangle shape this means light will be cast up as well as down.  

If you only and the light to be cast down then change the Light Distribution Type to Uniform Diffuse.  You will need to increase the intensity as the effect of the light will essentially be halved.

Uniform Diffuse

Uniform diffuse distribution casts diffuse light in one hemisphere only, as if the light were emitted from a surface.  If there is no surface or shape to the light there will be no light so you need to set the Emit Light (Shape) to a shape other than Point.

Spot Light

For Spot Light, the light shape must be a Point Light.  If for example you change Emit Light (Shape) of the Rectangle light above to Spot Light you will receive a warning that FireRender only supports point lights.  You will need to change the light shape back to a point.

Once you have adjusted the Emit Light from (Shape) back to Point, the spot light can be used like a normal spot light.  The Distribution (Spotlight) parameters will appear and allow you to modify the Hotspot/Beam and Falloff/Field.

Photometric Web

Photometric Web or IES lights are yet to be supported by AMD FireRender.  Their integration is in progress and will be coming soon.

A photometric web is a 3D representation of the light intensity distribution of a light source. This directional light distribution information is stored in a photometric data file in the IES format using the IES LM-63-1991 standard file format, or in the LTLI or CIBSE formats for photometric data. You can load photometric data files provided by various manufacturers as web parameters. In viewports, the light object changes to the shape of the photometric web you choose.
