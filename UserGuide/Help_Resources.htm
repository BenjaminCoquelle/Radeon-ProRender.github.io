Help Resources

The are a number of resources where you can get help with using the AMD FireRender plug-in.  

User Guide

This User Guide provides a Chapter based view of the AMD FireRender's capabilities as well as an Index and Search facility to quickly be able to find any particular topic on which you require support.

Many of the topics covered in the User Guide are also covered in Video Tutorials that we have provided below. The User Guide provides a quick access to the information you need, where as the Video Tutorials walk you though the workflow and every step of the process you need to produce quality AMD FireRender results.

Video Tutorials

Video Tutorials have been provided to enable you to quickly learn fundamental and advanced approaches to using the AMD FireRender plug-in. Currently these videos include:

  • Tutorial 1:  Quick start video; walk through of how to get setup, how to use the material library, quickly set up lights, and set up rendering.

  • Tutorial 2: Using the car asset that is available for download from the Download page.  Import geometry, setup environment, add additional lights, setup camera, render and save images.

  • Tutorial 3: Creating physically accurate materials with AMD FireRender in Autodesk 3ds Max - glass and physical properties.

  • Tutorial 4: Creating realistic materials with AMD FireRender in Autodesk 3ds Max.

Forums

The Autodesk Forum has a wealth of discussion and topics regarding 3ds Max as well as specific topics regarding AMD FireRender.  For feedback and support please feel free to post to this forum.
