Assign and Edit Materials

Assigning Materials

The car we imported has materials already assigned to it's parts.   We are going to remove some of these materials so that we can go through the process of assigning others.  Click on the car body which should highlight Nissan-GTR -> HDM_070 object in the scene hierarchy.  Type $ .mat = null in the 3ds Max script commend line.  

To remove the materials for any other objects repeat this process.  Or, you can leave the current materials that are assigned and overwrite them when you assign a new material.

To display the Materials Library open the Slate Material Editor either by using the Rendering -> Material Editor -> Slate Material Editor menu option, clicking the Material Editor button or hitting the M key.

Navigate to the FireRender_material libraries at the bottom of the left scroll bar in the Material Editor.

 

Select material library FireRender_metallic_paint.mat and in the dropdown select a material.  We will be using Car_Paint_Blue.  Drag the material into the workspace, click the body of the car if its not already selected, then right click over the material and select Assign Material to Selection to assign it to the car's body geometry.

Select the car tires and find FireRender_Rubber.mat in the material editor, in the drop down select Rubber_SmoothMatte and assign it to the tires.

Select the rims and under the FireRender_Metal.mat material library select the Chrome material and assign it to the rims.

 

Select the plastic parts (including the interiors).  Under the FireRender_Plastic.mat material library select Plastic_Matt_Black and assign it to the plastic parts.

Select the windscreen and all the glass components.  Under the FireRender_Glass.mat material library select the Glass_Solid_Clear material and assign it to the windscreen.

 

Select the lights.  Under the FireRender_emmisive.mat material library select the Emmissive_Flourescent_White material and assign it to the Light geometry.

Lastly select the floor geometry.  Under the FireRender_Concrete.mat material library select the Concrete_WornBare_SmoothBump material and assign it to the floor.  We will be adjusting this material to suit our needs next.

Editing Materials

We are now going to change the concrete floor so it looks more like a car dealers display floor.    The current concrete looks a little dull so let's give it a bit more polish.

 

First remove the bitmap from the roughness input of the FireRender microfacet material.  We will be leaving it as a smooth, semi reflective surface.

Change all bitmaps tiling to 100.  You can play around with the tiling until you get something that looks like a polished showroom floor.  

Select the blend material which is the main material and change the weight to 0.5.
