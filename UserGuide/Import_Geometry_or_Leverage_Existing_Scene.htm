Import Geometry

In this chapter we will look at importing geometry, setting up HDRI lights, assign some basic materials, add lights and finally end off with a rendered image.

For the rest of this chapter we'll be using the car scene from the samples included on the Download page. Details on how to find the AMD FireRender web site and Download page can seen in the Install, License, and Run FireRender for 3dsmax chapter.  

Download, unzip the Vehilce-2016-04-18.zip file and save the scene in a convenient location.  Open a new scene in 3ds Max.  We'll be using import and merge to take one of the cars from the sample scene into our new scene.

Select Import -> Merge from the 3ds Max MAX file management drop down.  Select the Verhicle_Booth_Demonstration.max scene from the sample download.

When the Merge dialog is displayed tick the Select Subtree check box and then scroll down the scene element to select the first Nissan-GTR line.  All the objects under that line will also be selected as a result. The click OK.

You should not have a new scene with the car object.  Center the car and save the scene.  You will see it is quite a large object when compared to the base grid of your new scene.
