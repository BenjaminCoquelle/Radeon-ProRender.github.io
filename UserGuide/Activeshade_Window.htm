Activeshade Window

If you want to see the render result as you change your scene yuo can set the render mode to ActiveShade mode.  Change the Target dropdown in the Render Setup to ActiveShade Mode.

Start a render and you will see as you move around, or move an object in the scene, that the render viewport will be updated with the changes you make.
